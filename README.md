# Resume Skill Extractor

**Project Overview:**  
This project extracts skills from uploaded resumes (PDF) and matches them with job descriptions. Built using Python, Flask, and NLP libraries.

**Features:**
- Upload resumes in PDF format
- Extract skills using NLP
- Match skills with job descriptions
- Display match score with visualization

**Tech Stack:**
- Python, Flask
- NLP: spaCy, NLTK
- PDF parsing: PyMuPDF
- Frontend: HTML, CSS, JavaScript

**Instructions to run:**
1. Clone the repo: `git clone <repo-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open browser at `http://127.0.0.1:5000`
